## fmeflow license refresh

Refreshes the installed license file with a current license from Safe Software.

### Synopsis

Refreshes the installed license file with a current license from Safe Software.

```
fmeflow license refresh [flags]
```

### Examples

```

  # Refresh the license
  fmeflow license refresh
```

### Options

```
  -h, --help   help for refresh
      --wait   Wait for licensing refresh to finish
```
